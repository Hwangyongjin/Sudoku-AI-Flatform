## 스도쿠 자동풀이 사이트  

### 프로젝트 설명
백트래킹 공부하다 삘받아서 만든 스도쿠 자동풀이 사이트
작업 기간: 2시간

### 주요 기능
1. 풀이 시작 클릭 시 스도쿠를 자동으로 풀이하여 모든 칸에 숫자를 채움
2. 풀이 시작 전 채우고 싶은 숫자를 미리 채우기 가능
3. 풀이가 불가능한 상태일 경우 에러 알
